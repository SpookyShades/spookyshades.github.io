# Spooks' ATS/ETS2 Repository
*Maps & Mileage Targets updated for 1.57*
- The following maps show what is visible in the map editor. The game is divided into a grid of sectors, wherein border sectors will be updated during development while the interior of upcoming DLCs remain hidden until purchase of the expansion.
- Mileage targets are the map editor's tool to automatically calculate the distances from a road sign to the city/location in question. Almost always, these targets are present in-game in some form, however accessible. Rare exceptions like Ashland, KS (ATS), or Å i Lofoten, NO (ETS2) may exist.
